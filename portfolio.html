Core Craft
Core Craft is an addon that i have had a big part in as a developer.
It has many entities/blocks/items that i've worked on, and many dynamic systems behind the scenes.

I am grateful to Ruben and Alyzix for giving me a shot, and it seems to have worked out pretty well.
Farhaven Reforged
Farhaven is a collection of many Minecraft SMPs.
Reforged was the 7th one in the series.

Farhaven holds a dear place in my heart, and i keep returning to it.

Farhaven 4, 5, the Soulful spinoff, and Reforged have all played a big role in my addon learning journey.
VeinCapitator
VeinCapitator is an addon that has been with me for a while now.

Though it is simple, it is very effective.

It has over 2 million downloads and is my most popular addon.
Simple Dynamic Lighting
Simple Dynamic Lighting is what it says.. well, kind of.

It's a fast, optimized, dynamic lighting that has almost a Java level of polish.

This also uses an API, which means that other packs can interact with it.
Java Essentials
Java Essentials is a more recent addon of mine.

This one is very complex mechanically, it's made for command users.

It brings a lot of functionality over from java, such as /worldborder, /team, /criteria, /tps, /speed, and more.
